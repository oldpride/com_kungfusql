This component provides an easy way to set up front-end query.
   The query result can be displayed on the page.
   The query result can be downloaded as a csv file.

To set up:
1. upload SQL files 
   upload the sql files to administrator/uploads/com_kungfusql

   there are some example sqls under uploads folder in the zip file

   note: administrator/uploads/com_kungfusql is outside module installation, therefore, will not
   be removed during uninstall.

2. Components->Kung Fu Sql, 
   add an entry to this sql file name

3. Menu->Main Menu, 
   add a new menu, 
   menu type: Kung Fu SQL
   select SQL file name
   set Params, up to 5
   change Debug mode if needed
   Parenet Menu
   Access
